News Explorer
Простое веб-приложение для поиска новостей через NewsAPI.
Пользователь вводит ключевое слово и получает 10 свежих статей.
Понравившиеся новости можно сохранить – они останутся в браузере даже после перезагрузки страницы.

Требования:

Установленный Node.js (версия 14 или выше)
npm (устанавливается вместе с Node.js)
API-ключ с сайта NewsAPI.org (бесплатный тариф даёт 100 запросов в день)

Как скачать проект:

Скачайте архив с проектом или выполните в терминале:

git clone <ссылка_на_репозиторий>
cd practica
Если вы скачали ZIP-архив, распакуйте его в любую папку и откройте терминал в этой папке.

Установка зависимостей:

В терминале (командной строке) находясь в папке проекта, выполните:

npm install
Эта команда установит все необходимые пакеты: Express, Axios, dotenv и Nodemon.
Настройка переменных окружения:
В папке проекта есть файл-пример .env.example.txt.

Скопируйте его содержимое в новый файл с именем .env (без .txt):

Откройте файл .env и замените значение NEWS_API_KEY на свой настоящий ключ, полученный от NewsAPI.org.
Также можно изменить порт (по умолчанию 3000).

Пример содержимого .env:

NEWS_API_KEY=ваш_ключ_сюда
PORT=3000

Запуск проекта:

Режим разработки (с автоматической перезагрузкой при изменении кода):
npm run dev

Обычный запуск:

npm start

После запуска в консоли появится сообщение:
Сервер запущен на http://localhost:3000
(порт может быть другим, если вы указали иной в .env).

Как пользоваться:

1. Откройте браузер и перейдите по адресу http://localhost:3000.
2. В поле «Search news...» введите слово (например, «JavaScript» или «технологии»).
3. Нажмите кнопку «Search».
4. Ниже появятся 10 новостей с заголовками, краткими описаниями и кнопкой «Save».
5. Чтобы сохранить новость – нажмите «Save». Она добавится в блок «Saved» внизу страницы и останется там после перезагрузки браузера.
